## Boutique Plugins
Boutique Plugins (aka BTQ Plugins) is a collection of wacky [OpenFX](https://github.com/AcademySoftwareFoundation/openfx) plugins for DaVinci Resolve (or any OFX host like VEGAS Pro) - creative effects like offset-and-wrap, sine wave distortions, colour adjustments & more. Several are original and some are ports of effects from video editing software like Kinemaster.
